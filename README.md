# DOOM風FPSゲーム

MiniPy3DR(pygame製の3Dレンダラ)のFPSツールキット `minipy3dr.fps` 
を使ったDOOM風FPSゲーム

## 実行環境の必要条件

* Python >= 3.10
* pygame >= 2.5
* minipy3dr >= 0.5.0 (`pip install minipy3dr`)

## 実行方法

```
python main.py
```

## ゲームの概要
* 主人公キャラクター豪華豚（ゴウカトン）をマウス操作により丸焼きにするゲームで，．．．
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーで豪華豚を操作し，スペースキー押下によるバーナーで豪華豚を丸焼きにする
* 豪華豚が10秒以上バーナーにあぶられたら，ゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* クールダウン機能（担当：あまり）：クリックした分だけ弾が打てるのを制限する機能
* 連射機能（担当：あまり）：マウス左クリックを長押しすると連射する機能
* リロード機能（担当：あまり）：rキーを押すとリロードする機能
* ショットガン機能（担当：あまり）：1クリックで複数の弾がでて散弾する機能
* 武器切替機能（担当：あまり）:キー1、2、3を押すと武器が切り替わる機能
* 反動とフラッシュエフェクト（担当：あまり）:撃った瞬間に反動とフラッシュが現れる機能
### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
